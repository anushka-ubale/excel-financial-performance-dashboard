# Excel Financial Performance Dashboard

## Project Overview
This project analyzes financial performance using an interactive Excel dashboard
to track revenue, expenses, profit, and departmental performance.

## Dataset
The dataset is stored in an Excel file and contains:
- Revenue and expense data
- Department-wise financial details
- Monthly and yearly performance metrics

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Key Insights
- Rent and utilities contribute the highest share of expenses
- Subscription and licensing are major revenue sources
- Some departments generate high revenue but lower profit margins
- Monthly profit trends highlight seasonal variations
